## TOKEN-FIRST VIBE ENGINEERING (Экономия токенов)

Для экономии лимитов ИИ-ассистента соблюдайте правила:
1. **Маршрутизация по мета-тегам:** В первой строке вашего запроса указывайте тег модели:
   - `#flash8b` — для поиска, документации, простых тестов и мелкого рефакторинга.
   - `#flash` — (по умолчанию) для написания функций, багфикса, анализа до 5 файлов.
   - `#pro` — для сложной отладки, проектирования, архитектурных решений.
2. **Контроль стоимости:** Используйте команды:
   - `/estimate "мой промпт"` — оценить стоимость запроса.
   - `/budget status` — проверить дневной баланс лимитов.
   - `/force` — запустить запрос в обход лимита Gateway (под вашу ответственность).
3. **Session Compression:** Когда чат перегружен, ИИ сохранит состояние в `SESSION_SUMMARY.md`. Скопируйте его и начните чистый чат для сброса контекста.
