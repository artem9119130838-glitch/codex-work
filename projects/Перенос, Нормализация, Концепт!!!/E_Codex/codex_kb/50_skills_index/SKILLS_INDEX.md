# Skills Index from Old Chats

These are reusable workflows discovered in old exports. Full source copies remain in `E:\Codex_Work\chat_exports`.

## Server / 1C / VPN

- `1c-postgres-triage`: distinguish SQL bottlenecks from 1C/rphost/network issues.
- `wireguard-1c-vpn-lockdown`: move 1C access to VPN and close public ports.
- `keenetic-gemini-dns-routing`: route Gemini through selected VPN without moving whole PC.
- `keenetic-amnezia-routing`: troubleshoot domain-based split routing.
- `n8n-vps-apache-deploy`: deploy n8n with Docker behind Apache/HTTPS on production-like VPS.

## Tenders / documents

- `astgoz-dom-automation`: automate AST GOZ through DOM links rather than coordinates.
- `excel-lot-consolidation`: merge and deduplicate tender lot workbooks.
- `pdf-quote-to-excel`: convert supplier PDF quote to Excel with formulas.
- `vfd-analog-assessment`: assess replacement compatibility.

## Local / productivity

- `weekly-inventory-audit`: weekly local computer file/program inventory.
- `google-calendar-deduplication`: clean duplicate birthday calendars.
- `google-calendar-voice-capture-router`: route voice-created events into calendars.

## WordPress

- `wordpress-acf-brand-snippet-triage`: diagnose ACF/template/page-meta mismatches.

## Rule

When a task matches one of these workflows, use the relevant domain file and source export before inventing a new process.
