# AGENTS.md — Инструкции для ИИ-агентов проекта "AI chats export"

Этот файл содержит регламенты и правила, которые ОБЯЗАН выполнять любой ИИ-агент, работающий с этим проектом.

---

## Ключевые регламенты и правила

1. **Локальность и переносимость (Относительные пути)**
   - Запрещено использовать абсолютные пути жесткого кодирования (например, `D:\Soft\...` или `C:\Users\Артем\...`).
   - Все пути в коде должны строиться относительно корня проекта.
   - Сырые данные лежат в [data/raw/](file:///C:/Codex_Personal/projects/AI%20chats%20export/data/raw).
   - Результаты пишутся в [data/done/](file:///C:/Codex_Personal/projects/AI%20chats%20export/data/done).

2. **Работа с кодировками файлов**
   - Все файлы Markdown, содержащие кириллицу, должны сохраняться в кодировке **UTF-8 с BOM** (utf-8-sig).
   - Логи и скрипты пишутся в кодировке UTF-8.

3. **Лимиты API и надежность работы с ИИ**
   - Бесплатный лимит Gemini API: **15 RPM** (запросов в минуту).
   - Внедряйте принудительную задержку не менее 12 секунд между запросами.
   - Используйте батчинг вместо мелкого дробления. Целевой размер батча для `gemini-2.5-flash` — около **300 000 символов** (75 000 токенов).
   - Обязательно оборачивайте запросы к API в блоки try-except с экспоненциальной задержкой (Exponential Backoff) для обработки ошибок 429, 503 и сбоев сети.

4. **Запрет ложного завершения (No False Completion)**
   - Агент не имеет права говорить "готово" без проведения автоматической верификации.
   - Запуск скрипта должен сопровождаться созданием/обновлением лога [data/done/report_log.txt](file:///C:/Codex_Personal/projects/AI%20chats%20export/data/done/report_log.txt) с детальной статистикой (обработано файлов, пропущено, ошибки).
   - Любые критические ошибки должны логироваться. Скрипт не должен падать на середине процесса из-за одного битого куска.

5. **Правила изменения кода**
   - Любое изменение логики фильтрации должно сначала тестироваться в режиме `dry-run` или на тестовом батче.
   - Исходный код скриптов размещается только в папке [src/](file:///C:/Codex_Personal/projects/AI%20chats%20export/src).
