## 3. Анализ ошибок и уроки (Lessons Learned)
- **Проблема прав pg_dump:** При создании резервных копий от имени postgres, pg_dump падал с Permission denied в папке root. Решение: chown postgres:postgres на целевую директорию.
- **Блокировка push из-за больших файлов:** GitHub отклонял отправку из-за файлов >100 MB (1Cv8.cf и tender-rag-api.zip). Решение: git reset --soft origin/master, git rm --cached <путь> и добавление в .git/info/exclude.
