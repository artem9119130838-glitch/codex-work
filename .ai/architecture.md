# Architecture C:\Codex_Personal

## Назначение проекта
Изолированный личный контур разработчика Артема для ведения личных проектов, планеров и канонической базы знаний (`codex_kb`).

## Структура контура
- `codex_kb/` — общая база знаний:
  - `00_global/` — системные правила (Security, Business, Operating Model).
  - `10_assets/` — серверы, локальный ПК, Keenetic роутер/VPN.
  - `20_domains/` — 1C, n8n, Bitrix, Tenders, Email.
  - `30_runbooks/` — инструкции и команды.
- `projects/` — личные проекты (исследования OData, Bitrix, инвентаризация).
- `scripts/` — служебные скрипты TOKEN-FIRST.

## Основные технологии
- Python, Bash, PowerShell.
- База знаний на Markdown.
- Git (личный репозиторий `artem9119130838-glitch/codex-work.git`).

## Ограничения и правила безопасности
- Никакого общего или коммерческого кода в этой папке.
- Запрещено хранить живые секреты в файлах. Использовать плейсхолдеры `${WIREGUARD_PRIVATE_KEY}`, `${ONEC_ODATA_PASSWORD}` и др.
- Сохранять файлы с кириллицей строго как UTF-8 с BOM.
